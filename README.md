# Docker Sync

## 仓库概述

同步 Dockerhub 镜像到阿里云私有仓库，目前支持的有以下镜像的部分版本：

- nginx
    - `latest`
    - `stable`

- mysql
    - `latest`
    - `8`
    - `8.0`
    - `5`
    - `5.7`

- redis
    - `latest`
    - `6`
    - `5`

- php
    - `8.1-cli`
    - `8.1-fpm`
    - `8.0-cli`
    - `8.0-fpm`
    - `7.4-cli`
    - `7.4-fpm`
    - `7.3-cli`
    - `7.3-fpm`
    - `7.2-cli`
    - `7.2-fpm`
    - `7.1-cli`
    - `7.1-fpm`
    - `7.0-cli`
    - `7.0-fpm`
